About

This webpage is a homage to the original 151 Pokemon designed by Nintendo. It will be a easy way for fans of the franchise to find information on the original 151 Pokemon. This site uses javascript to grab information from the Poke API and then wrap the information into the webpage. Using HTML5 & CSS, the website will eventually boast impressive visual effects as well as clean formatting. Ultimately, it will be a simple, effective and clean website built for a specific purpose to gather attributes about specific pokemon.

Current Status: Work In Progress
